# 智能图书推荐系统后端项目概览

## 项目基本信息

这是一个基于 **Spring Boot** 开发的智能图书推荐系统后端项目，采用模块化设计，提供完整的图书管理和个性化推荐功能。

## 核心技术栈

- **框架**: Spring Boot 2.6.13
- **数据库**: MySQL 8.0
- **ORM**: MyBatis-Plus 3.5.3
- **缓存**: Redis
- **认证**: JWT + Spring Security
- **API文档**: SpringDoc OpenAPI (Swagger)
- **JDK版本**: Java 8

## 主要功能模块

### 1. 认证模块 (`/api/auth`)
- 用户注册和登录
- JWT token认证

### 2. 图书模块 (`/api/books`)
- 图书列表和分页查询
- 多条件筛选（分类、主题、年份等）
- 图书详情查看
- 图书搜索

### 3. 推荐模块 (`/api/recommend`) ⭐
- **首页推荐**: "猜你喜欢"个性化推荐
- **新书推荐**: 新书抢先看
- **热门推荐**: 热门排行榜
- **主题推荐**: 主题漫游
- **相似推荐**: 相似图书推荐
- **协同过滤**: "用户也读过"推荐
- **推荐追踪**: 曝光、点击、反馈记录

### 4. 用户交互模块 (`/api/interaction`)
- 评论管理（增删改查）
- 收藏功能
- 借阅申请

### 5. 搜索模块 (`/api/search`)
- 图书全文搜索

### 6. 管理员模块 (`/api/admin`)
- 用户管理
- 图书管理
- 推荐系统管理

## 数据库设计

核心实体包括：

### 用户体系
- **User**: 用户基本信息
- **Role**: 角色信息
- **UserRole**: 用户角色关联
- **UserPreference**: 用户偏好
- **UserInterestGuide**: 用户兴趣引导

### 图书体系
- **Book**: 图书基本信息
- **BookTopic**: 图书主题
- **Booklist**: 书单
- **BookView**: 图书浏览记录
- **BookRelation**: 图书关系

### 交互体系
- **Rating**: 评分
- **Comment**: 评论
- **Favorite**: 收藏
- **BorrowRecord**: 借阅记录
- **Feedback**: 反馈

### 推荐体系
- **RecommendExposure**: 推荐曝光记录
- **RecommendClick**: 推荐点击记录
- **RecommendReason**: 推荐理由
- **RecommendStrategy**: 推荐策略
- **RecommendAnalytics**: 推荐分析数据

## 项目结构

```
backend/
├── src/main/java/org/example/backend/
│   ├── BackendApplication.java      # 启动入口
│   ├── common/                       # 公共模块（异常处理、工具类等）
│   ├── config/                       # 配置类（Security、Redis、CORS等）
│   ├── dto/                          # 数据传输对象
│   ├── entity/                       # 数据库实体
│   ├── modules/                      # 功能模块实现
│   └── vo/                           # 视图对象
├── src/main/resources/
│   ├── application.yml               # 配置文件
│   └── data_library126_db.sql        # 数据库初始化脚本
└── pom.xml                          # Maven依赖配置
```

## 关键配置

- **服务端口**: 9090
- **JWT过期时间**: 24小时
- **文件上传路径**: ./uploads
- **API文档**: 访问 `/swagger-ui.html` 查看

## 启动方式

1. 使用Maven构建：`mvn clean package`
2. 运行JAR文件：`java -jar target/backend-0.0.1-SNAPSHOT.jar`
3. 或直接运行Spring Boot应用

这是一个功能完善的毕业设计项目，涵盖了用户管理、图书管理、个性化推荐、用户行为分析等核心功能，采用了现代的Java技术栈和架构设计，适合作为智能推荐系统的学习参考。
